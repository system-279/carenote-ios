録音からドキュメント出力までのEnd-to-Endフロー

録音からドキュメントができるまで

ボタンひとつで録音が始まり、AIが自動で文字起こしし、ドキュメントとして出力されます。インターネット接続が不安定でも、接続が回復すれば自動で処理が再開されます。

  1. 録音

    高品質なM4A形式で録音。端末内に即座に保存されるので、録音データが失われる心配はありません。

  2. テンプレート選択

    録音後、AI文字起こしの出力形式を選びます。「議事録」「訪問記録」など、用途に合わせたテンプレートが用意されています。

  3. クラウドへアップロード

    インターネットに接続されると、録音ファイルがGoogle Cloudに安全にアップロードされます。

  4. AI文字起こし

    Google Gemini 2.5 Flash が音声を分析し、テキストに変換します。介護現場の専門用語にも対応しています。

  5. 結果の保存・表示

    文字起こし結果が端末とクラウドの両方に保存され、録音一覧から確認できます。

  6. Google Docs へエクスポート（任意）

    文字起こし結果をGoogle Docsに出力。タイトル・メタ情報・本文が自動フォーマットされたドキュメントが作成されます。

オフラインでも安心

📡

自動キューイング

通信できない場合、処理は自動的にキューに保存されます。手動での再送信操作は不要です。

🔄

自動再試行

一時的なエラーが発生しても、最大3回まで自動的に再試行します。間隔を空けて負荷を分散させます。

📶

ネットワーク監視

通信状態を常に監視し、接続が復帰した瞬間に自動的にアップロードを開始します。
