Security

認証・セキュリティ

WIF認証フロー、ロールベースアクセス制御、Auth Blocking Functionによるホワイトリスト管理

3つのセキュリティ原則

🔑

鍵を持ち歩かない

従来のアプリでは「秘密鍵」をアプリに組み込む必要がありましたが、CareNoteではWIF（Workload Identity Federation）を採用。アプリに秘密情報を含めず、毎回一時的な「通行証」を発行する仕組みです。

📦

招待制アクセス

管理者がメールアドレスを登録した人だけがサインインできます。未登録のメールでサインインしようとすると、サインイン自体がブロックされます。

👥

ロール（役割）管理

管理者（admin）はメンバー管理や設定変更が可能。一般ユーザー（user）は録音と文字起こし機能のみ。役割はサインインのたびに自動更新されます。

サインインの流れ

  1. Google アカウントでサインイン

    お使いのGoogleアカウントを選択するだけ。パスワードの管理はGoogleにお任せです。

  2. ホワイトリスト照合（自動）

    サインイン直前に、メールアドレスがホワイトリストに登録されているか自動確認。未登録の場合はサインインが拒否されます。

  3. 役割の自動設定

    ホワイトリストに登録された役割（管理者 or 一般）が自動的に適用されます。管理者が役割を変更すれば、次回サインイン時に自動反映されます。

  4. 安全な通行証の発行

    クラウドサービス（録音保存・AI文字起こし）を使うための「一時通行証」が自動発行されます。有効期限は約1時間で、自動更新されます。

Auth Blocking Function (ADR-005)

Firebase Cloud Functions v2 の beforeUserSignedIn を使用。毎回のサインインで custom claims を動的に設定。

// functions/index.js - 簡略化
exports.beforeusersignedin = onCall(
  { ...beforeUserSignedIn },
  async (event) => {
    const email = event.data.email;
    const snapshot = await db.collectionGroup('whitelist')
      .where('email', '==', email).limit(1).get();

    if (snapshot.empty) {
      throw new HttpsError('permission-denied', 'Not whitelisted');
    }

    const { role } = snapshot.docs[0].data();
    const tenantId = snapshot.docs[0].ref.parent.parent.id;

    return {
      customClaims: { tenantId, role }
    };
  }
);

Firestore Security Rules

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // テナントメンバー判定
    function isTenantMember(tenantId) {
      return request.auth != null
          && request.auth.token.tenantId == tenantId;
    }

    // 管理者判定
    function isAdmin() {
      return request.auth.token.role == 'admin';
    }

    match /tenants/{tenantId} {
      // テナントメンバーのみ読み取り
      allow read: if isTenantMember(tenantId);

      match /clients/{clientId} {
        allow read: if isTenantMember(tenantId);
        allow write: if isTenantMember(tenantId) && isAdmin();
      }

      match /recordings/{recordingId} {
        allow read: if isTenantMember(tenantId);
        allow create: if isTenantMember(tenantId);
        allow update: if isTenantMember(tenantId);
      }

      match /whitelist/{entryId} {
        allow read: if isTenantMember(tenantId);
        allow create, delete, update: if isTenantMember(tenantId) && isAdmin();
      }
    }
  }
}
